# 自治体PDF攻略（pdf-kouryaku）

ケアマネジメント等の対人援助職が、自治体からダウンロードした「印刷して手書き」前提のPDF様式を、ブラウザ上でそのまま入力してPDF出力できる無料ツールです。

**公開URL：** https://kat-t00.github.io/pdf-kouryaku/

インストール・会員登録は一切不要。リンクを開いて、お手元の自治体PDFを読み込ませればすぐ使えます。

---

## 主な機能

### ① テンプレート作成
- PDFを読み込み、入力してほしい範囲をドラッグで囲むだけで項目を登録
- 項目の種類は4種類：テキスト（複数行・文字配置対応）／マス目（1文字ずつの入力欄）／チェック／丸で囲む
- 項目のすぐ横に設定ポップアップが出て、実際のPDFの上に重ねたプレビューを見ながら文字サイズ・配置を調整可能。画面の他の場所をクリックすると自動保存されて閉じる
- ドラッグ移動・リサイズ・複製・削除、矢印キーでの微調整、スマートガイド（吸着）、元に戻す・やり直しに対応
- 「📋 コピーして新規作成」で、似た様式を作る時に項目をまるごと使い回し可能
- 保存済みテンプレートは名前で絞り込み検索・折りたたみ表示に対応。編集中のテンプレートは一覧から直接「上書き保存」も可能
- 全テンプレート／個別テンプレートのバックアップ・読み込みに対応

### ② 入力・出力
- 保存済みテンプレートを選んでPDF上の入力欄に直接入力、Tabキーで画面上の並び順に移動
- チェック・丸囲みはクリックでON/OFF切り替え
- 「④ 入力単語リスト」に登録した候補値をワンクリックで反映
- 利用者様ごとに入力内容をファイル保存・再開可能（対応ブラウザは同じファイルへの上書き保存にも対応）
- 「PDFを出力してダウンロード」または「新しいタブで開く（印刷用）」

### ③ 修正モード
- 既に文字が入った（去年提出した、など）PDFの一部だけを直したい時に使用（テンプレートとしては保存されません）
- 文字の書き直し・チェック追加・丸で囲むに対応。操作感はテンプレート作成モードと共通です

### ④ 入力単語リスト
- 氏名・主治医名・医療機関名など、繰り返し使う値を項目名ごとにストック
- 候補が1件だけの項目は自動入力、複数ある項目は入力欄横の「▾」ボタンから選択

---

## 動作環境

- 対応ブラウザ：Chrome / Edge / Safari などの最新ブラウザ
- サーバー不要、データは基本的にお使いの端末（ブラウザ）内に保存されます
- 個人情報を含むデータを外部に送信する機能はありません（完全にブラウザ内で動作）
- テンプレート・入力単語リストはブラウザ（端末）ごとの保存です。複数端末・複数人で共有したい場合は「📤 バックアップ」「💾 この入力内容を保存」でファイルに書き出し、手動で受け渡してください

---

## 使い方

1. 公開URLを開く
2. 「① テンプレート作成」で自治体PDFを読み込み、入力してほしい範囲をドラッグで囲んで項目を作る
3. 自治体名・様式名を入力して「テンプレートを保存」
4. 「② 入力・出力」で保存したテンプレートを選び、PDF上の入力欄に入力
5. 「PDFを出力してダウンロード」または「新しいタブで開く（印刷用）」

画面右上の「❓ 使い方ガイド」に、操作方法の一覧をまとめています。

---

## 既知の制限

- 入力単語リストとの対応づけは「項目名の完全一致」でしか行えません（表記ゆれには非対応）
- 入力・出力モードの自動保存は「作業中の一時的な控え」であり、正式な保管場所ではありません。確実に残したい入力内容は都度「💾 この入力内容を保存」でファイルに書き出してください
- 同じファイルへの上書き保存はChrome・Edgeなど対応ブラウザのみです。Safari・Firefox等では、これまで通り毎回新しいファイルとしてダウンロードされます

## 開発について

Claude Code（Anthropic）と共に開発。バニラHTML/CSS/JavaScript製で、ビルド不要。PDF表示に[pdf.js](https://mozilla.github.io/pdf.js/)、PDF書き込みに[pdf-lib](https://pdf-lib.js.org/) + [@pdf-lib/fontkit](https://github.com/Hopding/fontkit)、日本語フォントにNoto Sans JP（SIL Open Font License）を使用し、すべてローカル配置（外部CDNへの通信なし）で動作します。[ジェノグラム・エコマップ作成アプリ](https://kat-t00.github.io/genogram-app/)・[住環境見取り図アプリ](https://kat-t00.github.io/mitorizu-app/)の姉妹アプリです。
