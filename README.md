# 自治体PDF攻略（pdf-kouryaku）

自治体からダウンロードした「印刷して手書き」前提のPDF様式を、ブラウザ上でそのまま入力してPDF出力できるようにするツールです。

- サーバー不要・ブラウザだけで完結（入力したPDFや個人情報はどこにも送信されません）
- 一度「テンプレート」として様式を登録すれば、次回からはそのテンプレートを選ぶだけで入力できます
- テンプレートはブラウザのlocalStorageに保存されます（PC・ブラウザを変えると引き継がれません）
- 画面右上の「❓ 使い方ガイド」からも、いつでも操作方法の一覧を確認できます

## 使い方

### 1. ローカルで開く

`index.html`をダブルクリックするだけで動きます（日本語フォントはJSファイルに埋め込み済みなので、ローカルサーバーを立てる必要はありません）。

もしローカルサーバー経由で開きたい場合は、フォルダ内で以下を実行してください。

```
cd shinsei_form_app
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

### 2. テンプレート作成モード

1. 自治体PDFを選択すると、プレビューが表示されます（最初は様式全体が見えるよう自動で「全体表示」の縮尺になります）
2. 入力してほしい範囲をドラッグで四角く囲むと、右側のサイドバーの「項目の設定」に種類・文字サイズの設定が表示されます（フローティングのポップアップではなく常設パネルなので、様式やプレビューに重なって隠れることはありません）。文字の大きさ（マス目の場合はマスの数・間隔も）を調整すると、実際のPDFの上に本物のフォントサイズでプレビューが重なって表示されるので、数値を想像せずに目で見て調整できます
   - **テキスト**：1行の自由入力。枠の幅からはみ出す長さの文字を入れても自動でフォントサイズが縮小されます。「複数行の入力を許可する」にチェックを入れると、入力タブでの見た目が複数行入力欄になり、改行を含めて入力できます。PDF出力時は枠の幅で自動的に折り返され、行数が枠の高さに収まらない場合はテキストと同様に自動で文字が縮小されます。「文字の配置」で左寄せ・中央寄せ・右寄せを選べます
   - **マス目**：「被保険者番号」のように1マスに1文字ずつ入る欄向け。マスの数と「マスとマスの間隔」を調整すると、プレビューの区切り線が実際のPDFの罫線に重なるので、それを見ながら合わせられます。入力した文字は自動でマスの中央に振り分けられます
   - **チェック**：✓を書き込みます。項目名は空欄でも構いません（未入力なら「チェック1」のように自動で名前が付きます）
   - **丸で囲む**：「はい・いいえ」「男・女」のような二択を○で囲む、日本の行政書式でよくある形式です。こちらも項目名は空欄で構いません
3. 項目ボックスは中央をドラッグで移動、右下の四角いハンドルをドラッグで大きさ調整、ダブルクリックで種類・文字サイズを編集できます。複製⧉・削除×ボタンは、密集した申請書で隣の項目と干渉しないよう常時表示はせず、**項目をクリックして選択した時だけ**枠の上に表示されます（何もない場所をクリックすると選択解除されます）。項目名も常時表示はせず、枠にマウスを乗せるとツールチップで確認できます（一覧で確認したい時は下記の「このページの項目一覧」をご利用ください）
   - PDFプレビュー左上の「－／＋／全体表示」でズームできます。普段は様式全体を見渡せる「全体表示」のまま使い、被保険者番号やチェックリストなど項目が密集した箇所だけ拡大すると位置合わせがしやすくなります。トラックパッドのピンチイン・ピンチアウトや、キーボードの「＋」「－」「0」（0で全体表示に戻る）でも操作できます
   - 項目をクリックして選択すると、矢印キーで1ptずつ細かく位置を動かせます（Shiftを押しながらだと5ptずつ大きく動きます）。マウスでのドラッグより細かい位置合わせがしやすいです
   - 項目をドラッグで動かしている時、他の項目の端や中央と揃うとピンク色の細い線が出て、その位置にピタッと吸着します（Canvaなどにあるスマートガイドと同じ考え方です）
   - 項目を選択した状態でCtrl+C（Macは⌘+C）でコピー、Ctrl+V（⌘+V）で貼り付けできます。貼り付けるたびに少し下にずれて増えていくので、似た項目を連続で複製したい時にも使えます。DeleteキーまたはBackspaceキーで選択中の項目を削除、Escapeキーで選択解除もできます
   - ごく小さい欄（狭いマス目や小さい印字欄）にも対応できるよう、項目は数ピクセル四方まで小さく作れます。小さく作りたい時は先にズームインしてから操作すると掴みやすいです
   - 左上の「↩️ 元に戻す」「↪️ やり直し」（Ctrl+Z／Ctrl+Y、Macは⌘+Z／⌘+Y）で、項目の追加・移動・リサイズ・削除・複製・設定変更をさかのぼれます
   - PDFのプレビューは縦に長いのでスクロールが必要になりますが、項目が画面外にスクロールされて見えなくなっても消えたわけではありません。右の「このページの項目一覧」で項目名をクリックすると、その項目までプレビューが自動でスクロールし、一瞬光って場所を教えてくれます
   - **チェック・丸囲みを何個も並べたい時**（例：利用中のサービスのチェックリスト）は、PDF上で1つずつドラッグして作るより、1つ作ったらダブルクリックして開く「項目の設定」パネルの「⧉ 複製」を使うのがおすすめです。クリックするたびに直下へ少し間隔を空けて複製され、そのまま次の項目の編集状態になるので、「⧉ 複製」を連打するだけでチェックリストを量産できます。小さい項目同士が密集した状態でPDF上を直接ドラッグすると、隣の項目に触れて動かしてしまうことがあるため、この方法の方が確実です
4. 複数ページのPDFは「次のページ」で切り替えて、ページごとに項目を追加できます
5. 自治体名・様式名を入力して**「テンプレートを保存」を必ず押してください**
   - 「テンプレートを保存」を押す前の作業内容も自動的に下書き保存されます（個人情報は含まれないPDF様式・項目位置のみ）。これはブラウザを誤って閉じてしまった時の**復旧用**で、ブラウザを閉じても次にテンプレート作成モードを開けば続きから復元されます
   - ただし**この自動保存だけでは「入力・出力」タブでは使えません**。テンプレート一覧に正式に登録され、他のタブからも使えるようになるのは「テンプレートを保存」ボタンを押した時点です。保存せずに他のタブに移動しようとすると確認のメッセージが表示され、その場でテンプレート名を入力して「今すぐ保存して移動」を押せば、タブを行き来せずそのまま保存・移動できます
6. 「📤 全テンプレートをバックアップ」で、これまで作った全テンプレート（PDF本体＋項目定義）を1つの.jsonファイルに書き出せます。1件だけ渡したい時は、各テンプレートの「📤 このテンプレだけバックアップ」を使ってください。「📥 バックアップから読み込む」で取り込む時は、ファイルに複数件入っていても一覧から必要な分だけチェックを入れて選んで取り込めます（個人情報にあたる入力単語リストは含まれません）
7. **他の自治体の似た様式を作る時**は、保存済みテンプレートの「📋 コピーして新規作成」から新しいPDFを選ぶと、元テンプレートの項目（種類・ラベル・文字サイズ・座標）がそのままコピーされた状態で始められます。多くの様式は項目の並びが似ているため、位置をドラッグで合わせるだけで済み、一から作るより早く仕上がります（コピー元とコピー先は別々のテンプレートとして保存されます）

テンプレート作成モードと同じ2カラムのレイアウトです（PDFプレビューが左、操作パネルが右のサイドバーにまとまっています）。

1. 右の「様式を選ぶ」パネルで保存済みテンプレートを選択（選ぶと最初は様式全体が見える「全体表示」の縮尺になります。テンプレート作成モードと同じく左上の「－／＋／全体表示」、トラックパッドのピンチ、キーボードの「＋」「－」「0」でズームできます）
2. PDFの上に表示された入力欄に文字を入力。入力欄はどこが入力できる場所か分かるよう薄いグレーの点線でそっと示していますが、読み込んだPDFの罫線を隠すほど主張はしません（白塗りは③修正モードの出力時だけの処理です）。項目名（「月」「日」「主治医」など）は欄の中には表示されません（狭い欄では特に見づらいため）。何の項目か確認したい時は、欄にマウスを乗せるとツールチップで見られます。チェック・丸囲みの項目は、実際に出力される✓や○そのものの見た目のマークになっていて、クリックでON/OFFを切り替えられます（未選択時は薄いグレーの点線、選択済みは黒の実線・黒い✓と、実際にPDFへ書き込まれる色に合わせています）。マス目タイプの項目は1マスごとに入力欄が分かれていて、テンプレート作成時に設定した文字サイズ・間隔がそのまま反映されます。1文字入れると自動で次のマスに移動し、Backspaceで前のマスに戻れます。日本語入力（IME変換）にも対応しており、変換確定時に複数文字が一度に入っても自動で後続のマスへ振り分けられます。Tabキーで次の入力欄に移動できます（項目を作った順ではなく、画面上の位置＝上から下・左から右の順に移動します）
3. 右下の「このページの入力状況」に、現在のページの全項目とその入力内容・チェック状態が一覧表示されます。項目名をクリックするとPDFプレビューがその項目までスクロールして一瞬光ります（チェック・丸囲みは種類の表示部分をクリックするとON/OFFも切り替えられるので、PDF上の小さい枠を正確にクリックしにくい時はこちらが便利です）
4. 「PDFを出力してダウンロード」または「新しいタブで開く（印刷用）」

テンプレート作成モードで今開いているテンプレートを編集・保存すると、入力内容を保ったまま様式が自動的に最新化されます。

入力内容は自動保存されます（作業中の一時的な控えとしてブラウザに保存）。ブラウザを誤って閉じても、次回このモードを開くと自動で復元されます。「この入力内容を保存」で.jsonファイルに書き出すと、ブラウザ内の一時控えは役目を終えてクリアされます。

**利用者様ごとに入力内容を保存する**：入力が終わったら（または申請の途中でも）「💾 この入力内容を保存」で.jsonファイルとして書き出せます。様式そのもの（PDF・項目位置）と入力した答えの両方が1つのファイルに入るので、そのまま利用者様の管理フォルダに保管しておけます。次回このアプリの「📂 保存した入力データを開く」でそのファイルを開くと、**同じidのテンプレートがまだ一覧に残っていれば、その最新版（後から追加した項目なども含む）が優先して使われ、入力済みの値はそのまま引き継がれます**。テンプレートが削除済み・別のPCなどで一覧に見つからない場合だけ、ファイルに保存時点の様式がそのまま埋め込まれているので、それを使って変わらず開けます。ファイル名に使う「利用者名など」の欄は空欄でも構いません。
- **上書き保存について**：Chrome・Edgeなど対応ブラウザでは、「📂 保存した入力データを開く」で開いたファイル、または一度「💾 この入力内容を保存」で保存先を選んだファイルに対しては、続けて「💾 この入力内容を保存」を押すと同じファイルにそのまま上書き保存されます（新しいファイルがどんどん増えていくことはありません）。保存先を保管フォルダの下に置いておけば、そのまま何度でも上書きして最新の状態を保てます。別のテンプレートを選び直したり、はじめから作り直したりすると、この結びつきは解除されて次回は改めて保存先を聞かれます。対応していないブラウザ（Safari・Firefox等）では、これまで通り毎回新しいファイルとしてダウンロードされます。

### 4. 入力単語リスト

氏名・主治医名・医療機関名など、様式に繰り返し出てくる項目名ごとに、候補となる値をいくつでもストックしておけます。「主治医名」のように担当利用者ごとに内容が変わる項目は、複数の候補を登録しておいて、その場でどれを使うか選ぶ形になります。

1. 「④ 入力単語リスト」の「＋ 項目を追加」で項目名を登録します。項目名欄はこれまでテンプレートで使った項目名が候補として出てくるので、1から手打ちしなくても選べます（一覧にない新しい名前も自由に入力できます）。候補値はいくつでも追加でき、チップの×で個別に削除、「この項目名ごと削除」でまとめて削除できます
   - 項目名は普段「▶ 主治医名　3件」のように折りたたまれた状態で並びます。登録が増えても一覧が縦に伸びすぎないようにするためで、クリックすると候補の中身が開きます
2. 入力・出力モードでテンプレートを選ぶと：
   - **候補が1件だけの項目**は「📋 候補が1件だけの項目を自動入力」で反映されます（すでに入力済みの欄は上書きしません）
   - **候補が複数ある項目**は自動入力されません。該当する入力欄を選択（クリック）している間だけ、すぐ右に「▾」ボタンが表示されるので、クリックするとその場に候補一覧が開き、選ぶだけで反映されます（候補のある項目が多い様式でも、選択していない項目のボタンは隠れているので様式の文字と重なりません）
3. 逆に、実際に入力した内容を「💾 入力内容を単語リストに追加」で候補として書き戻すこともできます（既存の候補は消えず、新しい候補として追加されます）

### 5. 修正モード

すでに文字が入った（去年提出した、など）PDFの一部だけを直したいときに使います。テンプレートとしては保存されません。操作感はテンプレート作成モードとほぼ同じです。

1. 修正したいPDFを選択（読み込むと最初は様式全体が見える「全体表示」の縮尺になります）
2. 直したい場所をドラッグで四角く囲むと、右側のパネルに設定（種類：文字を書き直す／チェックを入れる／丸で囲む）が表示されます
3. 「文字を書き直す」を選んだ場合は差し替える文字を入力（空欄なら白抜きだけで消すだけになります）。テンプレート作成モードと同じく、文字の大きさ・複数行・配置（左寄せ／中央寄せ／右寄せ）を設定でき、実際の見た目をその場のプレビューで確認しながら調整できます
   - **チェックを入れる**を選んで追加すると、その場所に入力・出力モードと同じ見た目の✓が表示されます。クリックするとON/OFFを切り替えられます（OFFの時はPDF出力時に✓を描かず、白抜きだけになります）
   - **丸で囲む**は「はい・いいえ」のように既にPDFに印刷されている文字を囲むための機能なので、白抜きはされません（文字を書き直す／チェックの時だけ、その場所を白く隠してから描画します）。こちらもクリックでON/OFFを切り替えられます。丸自体は枠の見た目が変わらないため、OFFの時は薄く表示してひと目で分かるようにしています
   - チェック・丸囲みのON/OFFは、PDF上の枠を直接クリックするほか、右下の「このページの修正箇所」一覧からも切り替えられます
4. 修正箇所は中央をドラッグで移動、右下ハンドルで大きさ調整、クリックして選択すると複製⧉・削除×ボタンが出ます（何もない場所をクリックすると選択解除）
5. 左上の「－／＋／全体表示」でズーム、「↩️ 元に戻す」「↪️ やり直し」（Ctrl+Z／Ctrl+Y）で操作をさかのぼれます。項目を選択した状態でCtrl+C／Ctrl+Vでコピー＆貼り付け、Delete/Backspaceで削除、矢印キーで位置の微調整もできます（テンプレート作成モードと同じ操作方法です）
6. 「PDFを出力してダウンロード」または「新しいタブで開く（印刷用）」

## 技術構成

- PDF表示：[pdf.js](https://mozilla.github.io/pdf.js/)
- PDF書き込み・出力：[pdf-lib](https://pdf-lib.js.org/) + [@pdf-lib/fontkit](https://github.com/Hopding/fontkit)
- 日本語フォント：Noto Sans JP（`lib/fonts/`、SIL Open Font License。ライセンス条文は同フォルダの`OFL.txt`参照）。`fetch()`に頼らずfile://でも読み込めるよう、`notosansjp_base64.js`としてbase64埋め込み済み
- PDF出力時はフォントを部分埋め込み（使う文字だけ抜き出す方式）にしていません。部分埋め込みだと、pdf-libの既知の不具合で使用文字数が多い時に一部の漢字が印刷・PDF上で表示されなくなる問題があったため。そのぶん出力PDFのファイルサイズはやや大きめ（1件あたり数MB程度）になりますが、文字が確実に表示されることを優先しています
- すべて`lib/`にローカル配置し、外部CDNへの通信なしで動作します
- テンプレートの項目データは読み込み時に自動で形式を補正します（開発中に項目のデータ形式を変更したことがあるため。古い形式で保存されたテンプレートを開いても壊れずに表示・編集できます）

## 制限事項（今後の改善候補）

- 1項目＝1行のテキスト入力のみ（複数行の折り返しには未対応。幅を超えた場合の自動縮小のみ対応）
- 入力単語リストとの対応づけは「項目名の完全一致」でしか行えない（表記ゆれには非対応）
- 入力・出力モードの自動保存は「作業中の一時的な控え」であり、正式な保管場所ではない（ブラウザのデータを消すと失われる）。確実に残したい入力内容は都度「この入力内容を保存」でファイルに書き出してください
